This folder contains all of the files that let Ollama models interact with the VLA via MCP.
Learn more about MCP (Model Context Protocol): https://modelcontextprotocol.io/docs/getting-started/intro

This MCP (specifically meant for local models rather than models accessible through the internet) is built on top of the Phosphobot API, which you can install and learn more about here: https://docs.phospho.ai/welcome
